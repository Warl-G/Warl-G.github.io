# Warl-G's Blog  
## 说明  
本博客以[Jekyll](https://jekyllrb.com/)技术支持，在[texture](https://github.com/thelehhman/texture)主题基础上添加部分功能

## 已加功能
* 分类索引页（Category）
* 标签索引页（Tag）
* 目录索引
* Footer说明
* 主界面：总字数和访问人次
* 文章独立页：文章字数和访问次数
* 百度统计  
* 国内部分博客入口
* 网页title logo
* 搜索引擎可搜索
	* 添加谷歌验证
	* 百度被gitpage屏蔽，需做国内镜像，不做了，直接投稿国内网站
* GitTalk评论系统
