# Warl-G's Blog  
## 说明  
本博客以[Jekyll](https://jekyllrb.com/)技术支持，在[texture](https://github.com/thelehhman/texture)主题基础上添加部分功能

## 增加
* 分类索引页（Category）
* 标签索引页（Tag）
* 目录索引
* Footer说明
* 主界面：总字数和访问人次
* 文章独立页：文章字数和访问次数
* 网页title logo
* 国内个人博客入口
* 百度统计
